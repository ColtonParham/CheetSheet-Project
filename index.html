HTML CHEETSHEET

Tag Name Description
<div> Divider The div will act as a divider between groups.
<h1> Header The Element for defining a header, and its size
<body> Body This element makes up the body of the page
<html> HTML This element makes up the encapsulator of the page
<nav> NAV This element serves as the navigation section of the webpage.
<ul> Unordered List This element lists the list items with no specific order
<ol> Ordered List This element has it so the list items print in a specific order.
<li> list This element acts as an item inside an Unordered List, or Ordered List.
<head> Header(2) This is the alternative way to add a header.
<p> Paragraph This element lets us introduce a paragraph

TABLE CHEATSHEET

Tag Name Description
<tbody> Table Body This will act as the body of the table.
<thead> Table head This will act as the head of the table.
<tr> Table Row Element This will act as the table row element inside of the table.
<td> Table Data Cells This will act as data cells within each tab.
